### Installation

1. Clone the repo
   ```sh
   git clone https://gitlab.com/odrassam/sky-angel-js
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### How to run in local

1. Follow the installation guide first

2. Run Json Server
   ```sh
   npm start-db
   ```
   ```sh
   yarn start-db
   ```

3. Run App
   ```sh
   npm start
   ```
   ```sh
   yarn start
   ```
